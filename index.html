Commande de base

echap = revenir en mode normal

:wq = enregister et quitter

:q! = quitter sans sauvegarder

i = inserer du texte avant le curseur

a = inserer du texte apres le curseur

dw = effacer un mot (placer curseur au debut du mot)

de = effacer le mots (nefacer pas le dernier caractere)

d$ = effacer la fin de la phrase (apres le curseur)

d2w = effacer les mots en majuscule

dd = effacer une ligne complette

2dd = effacer deux lignes

(nombre)dd = effacer autant de ligne que necessaire

P = place aprés le curseur ce qui vien d'etre effacer

2w = deplacer le curceur de deux mots

3w = deplacer le curseur de trois mots

0 = mettre le curseur au debut de la ligne

r(x) = remplacer le caractere sous le curseur par (x)

R = remplacer plusieurs caractére a la fois

ce = remplace le mots complet

y = copier

p = coller

u = annuler la derniere ligne de code

U = récuperer toute une ligne

G = deplacer a la fin du fichier

gg = deplacer au debut du fichier

CTRL-G = affiche la position dans le fichier

(numeros de ligne) G = se rendre a la ligne souhaiter

/(texte) = pour rechercher le texte en question

n = rechercher le meme texte

N = rechercher le meme texte dans le sence contraire

? au lieu de / = pour rechercher une phrase a l'opposé

% = pour se deplacer sur les "( ) , [ ] , { }"

:(numero),(numero)/ancien/nouveau/g = remplace les mots voule par autre chose sur les ligne selectionner

:%s/ancien/nouveau/g = remplacer tout les mots du fichier par d'autres

:W fichier = enregistrer les modif du fichier

:! rm fichier = effacer le fichier selectionner

v mouvement(direction) = selection de ligne

:(text selectionner)w (fichier) = enregistrer le fichier selectionner

:r (fichier) = inserer le text enrigister

o = ouvre une commande a la ligne et inser un texte

e = aller a la fin du mot (appuiller sur a pour ajouter text)

:edit $HOME/.vimrc = script de demarrage
